#3 

Longest Substring Without Repeating Characters

难度中等
Given a string, find the length of the longest substring without repeating characters.

思路：莫名其妙地卡了很久。。找最长的无重复连续子串。无重复所以删去重复元素，连续所以逐个遍历、逐个删除。
class Solution {
    public int lengthOfLongestSubstring(String s) {
        int max = 0;
        int len = s.length(), start = 0;
        int count = 0;
        Set<Character> set = new HashSet<>();
        for (int end = 0; end < len; end++){
            char ch = s.charAt(end);
            while (set.contains(ch)){
                set.remove(s.charAt(start++));
                count--;
            }
            set.add(ch);
            max = Math.max(max, ++count);
        }
        return max;

    }
}

class Solution {
    public String longestPalindrome(String s) {
        int len = s.length();
        if (len < 2) return s;

        int max = 0;
        String ans = new String();
        char[] array = s.toCharArray();

        for (int i = 0; i < len; i++){
            for (int j = i; j < len; j++){
                if (validate(array, i, j) && (j-i+1) > max){
                    max = j- i + 1;
                    ans = s.substring(i, j+1);
                }
            }
        }
        return ans;
    }
    private boolean validate(char[] arr, int i, int j){
        while (i <= j){
            if (arr[i++] != arr[j--] )
            return false;
        }
        return true;
    }
}

#46 Permutations

思路：First, tring to understand that the process of 'n' elements permutations could be presented by a tree. The height of the tree is 'n'. 

用程序的思维来看待全排列时，可以用两种思路对其进行描述。

第一种，每次取出一个尚未被用过的元素，向下穷尽所有可能后将状态回溯。
这种思路全程只需要一个状态数组(used[])和一个记录当前路径的变量(path)。

采用这种思路面临着几个问题。一，如何保证每次取出的都是未被使用过的元素？答案是声明一个状态数组去保存每个元素的使用状态。每次在处理某个树的节点时，遍历状态数组寻找未使用的元素，找到后将其加入当前已有的路径中，再向下搜索。向下搜索完成后，回溯到上一个节点。回溯的具体操作是，将当前节点改为可用，并将当前路径的最后一个元素去掉。

每次搜索到的路径长度等于n时，表示找出了一个全排列的可能性，将其加入最后的答案中。
